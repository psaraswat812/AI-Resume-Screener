# AI Resume Screener

An AI-powered Resume Screening and ATS Optimization platform built using FastAPI, Next.js, PostgreSQL, Machine Learning, and Google Gemini AI.

## Features

* Resume Upload (PDF & DOCX)
* ATS Score Analysis
* Skill Extraction
* Missing Skill Detection
* AI Resume Rewriting
* AI Resume Chatbot
* Resume Comparison
* User Dashboard
* Recruiter Dashboard
* JWT Authentication
* PostgreSQL Database Integration

## Tech Stack

### Frontend

* Next.js
* React
* Tailwind CSS

### Backend

* FastAPI
* SQLAlchemy
* PostgreSQL
* JWT Authentication

### AI & Machine Learning

* Google Gemini AI
* Scikit-Learn
* NLP-Based Skill Extraction

## Installation

### Backend

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
npm install
npm run dev
```

## Environment Variables

Backend `.env`

```env
DATABASE_URL=
SECRET_KEY=
GEMINI_API_KEY=
```

Frontend `.env.local`

```env
NEXT_PUBLIC_API_URL=
```
